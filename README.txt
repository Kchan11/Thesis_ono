知能情報工学科 卒業論文 LaTeX スタイルファイルセット 2023

卒業論文に関連した LaTeX2ε用のスタイルファイルを用意しました.
使い方などは abst.pdf を参考にしてください．

クラスファイル, スタイルファイル
  AIabst.cls:       概要用クラスファイル
  AIcover.sty:      表紙用スタイルファイル
  AIthesis.sty:     論文用スタイルファイル
  personal.sty:     個人情報用スタイルファイル

サンプルファイル
  abst.tex:         概要サンプル(兼 概要, 表紙ファイルの使い方)
  main.tex:         論文サンプル
  personal.tex:     個人データサンプル

ドキュメント
  abst.pdf:         サンプル(abst.tex)の PDF 版
  main.pdf:         サンプル(main.tex)の PDF 版
